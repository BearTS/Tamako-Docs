## Ticket Support System
| Command | Description |
| ------- | ----------- |
| `t!add` | Add users to a ticket
| `t!archive` | Archive a ticket. Sends the user a text transcript of the ticket and deletes the channel

| `t!post` | Post a message that users can react to in order to open tickets
| `t!remove` | Remove users from a ticket
| `t!rename` | Changes a ticket name. If no hid is given, renames the current channel ticket
| `t!setup` | Set up the server for Ticket System by Tamako.
| `t!ticket` | Get all commands or help related to ticket support System
| `t!ticketlimit` | Set server limit for how many tickets users can have open at the same time(default is 5)
| `t!unbind` | Unbind the ticket starter reaction from a custom message
| `t!userlimit` | Set server limit for how many people can be added to a ticket via the add command.(default is 10)
